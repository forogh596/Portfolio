Projet universitaire

Création d’un site web pour un client

Contexte

Projet universitaire réalisé en équipe de quatre étudiants consistant à concevoir un site web à partir des besoins exprimés par un groupe d'élèves de troisième jouant le rôle de clients.

Objectif

Comprendre les attentes du client, organiser les informations importantes et concevoir une interface web claire, cohérente et adaptée aux utilisateurs.

Ce que j’ai réalisé

  • Analyser les besoins du client et identifier les fonctionnalités attendues
  • Structurer le contenu du site pour rendre l’information facile à comprendre
  • Concevoir une interface simple, moderne et agréable à utiliser
  • Réaliser les pages du site avec HTML et CSS
  • Présenter le projet et expliquer les choix réalisés

Technologies

HTML, CSS, conception d’interface

Compétences développées

Analyse de besoins, communication, organisation, développement web
