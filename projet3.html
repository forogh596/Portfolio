Projet universitaire

Création d’une base de données

Contexte

Projet universitaire réalisé en binôme, portant sur la conception et la création d’une base de données relationnelle répondant à un besoin précis.

Objectif

Concevoir une structure de données cohérente et organiser les informations afin d’assurer une gestion efficace et fiable des données.

Ce que j’ai réalisé

  • Analyser les besoins afin de définir les données nécessaires au projet
  • Modéliser les relations entre les différentes entités du système
  • Créer et organiser les tables d’une base de données relationnelle
  • Écrire des requêtes SQL pour interroger, filtrer et manipuler les données
  • Structurer les informations pour garantir une gestion claire et optimisée

Technologies

SQL, base de données relationnelle

Compétences développées

Modélisation de données, requêtes SQL, analyse et organisation des données
